#### App平台差异
- app平台 font-weight 样式不支持继承  
- app-android平台不支持任意值粗细，仅支持 normal 和 bold 两种值，大于等于 500 时按 bold （加粗）显示，小于 500 时按 normal（正常）显示
- app-android平台，部分自定义字体不支持设置 font-weight
