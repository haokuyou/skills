- App平台
	+ HBuilderX3.92及以前版本默认值为0px，HBuilderX3.93+版本调整默认值为thin
	+ HBuilderX4.0+版本调整默认值为medium
- Web端
	+ Android平台Chrome浏览器或内置Webview中实际默认值不是medium，是根据设备自动计算的介于thin和medium中间的值
